# Business Workflow

## Purpose
This document defines the business logic, user journey, and monetization rules of the product.
It describes WHAT the product does and WHY, without technical implementation details.

This product exists to help non-technical users create a simple, trustworthy online presence.

---

## Target User
- Non-technical individuals
- Small businesses
- Freelancers
- First-time website creators

Users want:
- Safety
- Simplicity
- Confidence
- A live website they can share

They do NOT want:
- Code
- Technical settings
- Design tools
- AI prompts

---

## Core Product Promise
"Create a complete online presence in minutes, without technical skills."

---

## Website Characteristics
- Single-page website only
- Static output: HTML + CSS
- No JavaScript in the generated site
- Always includes:
  - Navbar
  - Footer

---

## User Journey (High-Level)

1. User creates an account
2. User starts a new project
3. User selects predefined options:
   - Website type
   - Theme
   - Primary goal
4. User provides simple inputs:
   - Brand name
   - Optional tagline
   - Optional logo (png/jpg/jpeg or no logo)
   - Optional text about their business
5. System generates a plan
6. User reviews and approves the plan
7. System generates the website ONCE (free users)
8. User previews and edits text
9. User publishes the website
10. 15-day live trial starts
11. User upgrades to keep the site live

---

## Free Plan Rules (STRICT)

Free users are allowed:
- 1 project only
- 1 planner run
- 1 website generation (no regeneration)
- Manual text editing (no AI)
- Publish the site
- 15 days of live hosting

Free users are NOT allowed:
- Regenerate the site
- Re-run the planner
- Change theme or layout
- Create another project
- Use custom domain
- Export files

---

## Paid Plan Rules

Paid users unlock:
- Multiple projects
- Limited regenerations (never unlimited)
- Custom domain
- Remove platform branding
- Keep site live beyond 15 days
- Version history (optional)

AI usage is always capped, even for paid users.

---

## Trial Rules

- Trial starts ONLY after publish
- Trial duration: 15 days
- During trial:
  - Site is fully live
  - No extra AI usage is granted
- After trial expires:
  - Site is paused
  - User must upgrade to reactivate

---

## Editing Rules

Users can:
- Edit text content at any time
- Fix typos
- Update wording

Users cannot:
- Edit HTML structure
- Edit CSS
- Add scripts
- Change layout or theme (free)

---

## AI Usage Philosophy

AI is a controlled system resource.
AI is not unlimited creativity.

AI is used only to:
- Plan site structure
- Generate initial HTML
- Generate initial CSS

All AI usage is limited, tracked, and gated.

---

## Admin Philosophy

Admin exists to:
- Support users
- Observe system health
- Prevent abuse

Admin does NOT:
- Bypass rules
- Execute AI freely
- Break the state machine

---

## Core Business Principles

- Simplicity over flexibility
- Safety over power
- Predictability over customization
- Trust over tricks
